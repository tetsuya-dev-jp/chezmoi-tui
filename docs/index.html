chezmoi-tui

The problem

chezmoi is powerful.
But CLI-only hides context.

You can't see what changed, what's managed, or which action is safe to run next — without chaining multiple commands.

CLI only

  • Run chezmoi status, then chezmoi diff, then decide
  • No way to see managed vs unmanaged at a glance
  • Dangerous actions run without guardrails
  • Batch operations require shell scripting

With chezmoi-tui

  • 3-pane layout: List + Diff/Preview + Log at once
  • Switch views instantly: Status / Managed / Unmanaged
  • Typed confirmation for destroy & purge
  • Multi-select with Space, batch execute from menu

Features

Everything you need,
nothing you don't.

3-Pane Layout

File list, diff/preview, and command log visible simultaneously. No more switching between terminal tabs.

Rich Diff Rendering

Hunk headers, line numbers, and color-coded status. Auto-loads diff when you select a file in status view.

Tree Navigation

Expand/collapse directories with h/l in managed and unmanaged views. Familiar vim-style navigation.

Batch Execution

Mark multiple files with Space, then run actions on all at once from the action menu. Clear marks with c.

Safety First

Dangerous actions require typed confirmation. Directory-wide add is blocked. Force flags prevent TUI deadlocks.

Persistent Config

Theme, keymap, and preferences saved to ~/.config/chezmoi-tui/config.toml. Customize once, keep forever.

Views

Three views. Full picture.

Switch instantly between status, managed, and unmanaged views. Each shows the right context and actions for that scope.

Safety

Built with guardrails.

Dotfiles are critical infrastructure. chezmoi-tui makes sure you never accidentally destroy them.

Typed Confirmation

destroy and purge require you to type the exact confirmation phrase. No accidental Enter key disasters.

Directory Add Blocked

Directory-wide add is blocked to prevent accidental bulk imports into chezmoi source state.

Edit Restricted

edit only works on managed files. No risk of opening untracked files through the wrong view.

Force + No-TTY

forget and purge run with --force --no-tty to prevent chezmoi from stealing the TUI's terminal.

Get started in seconds.

Requires Rust 1.93+ and chezmoi in PATH. macOS & Linux.
